PLG Ideas

Privacy Policy

Last updated: April 29, 2026

What PLG Ideas does

PLG Ideas is a Chrome extension that reads the page you are currently viewing and uses the Anthropic Claude API to generate product-led growth experiment ideas tailored to that page.

Data we collect — none

PLG Ideas does not collect, transmit, store, or share any personal data with the developer. There are no analytics, no tracking pixels, and no servers operated by this extension.

Data stored on your device

The following is stored locally in your browser using chrome.storage.local and never leaves your device except as described below:

WhatWhy
Anthropic API key Required to call the Claude API on your behalf
Product context Your product name, target customer, and growth challenge — used to personalize experiment suggestions
Experiment history Up to 20 past results so you can revisit previous experiments

You can clear any of this data at any time via the extension's Settings page.

Data sent to Anthropic

When you click "Generate Ideas," the extension sends the following to the Anthropic API:

  • The URL and title of the active tab
  • Visible text from the page (headings, CTA button text, navigation labels, body copy excerpt)
  • Your product context, if you have set it

This data is sent directly from your browser to Anthropic's servers using your own API key. It is subject to Anthropic's Privacy Policy and Usage Policy. PLG Ideas does not intercept or store this exchange.

Permissions used

PermissionWhy it's needed
activeTab To read the page you want to analyze
scripting To inject the content script that extracts page data on demand
storage To save your API key, product context, and history locally
api.anthropic.com To call the Claude API

Third parties

The only third party involved is Anthropic. No other services, SDKs, or analytics libraries are included.

Children

This extension is not directed at children under 13 and does not knowingly collect information from them.

Changes

If this policy changes materially, the updated version will be posted at this URL with a revised date.
